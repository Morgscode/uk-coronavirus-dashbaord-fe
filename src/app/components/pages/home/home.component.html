Coronavirus in the UK

An Independant dashboard for UK wide Coronavirus figures.

This website provides numeric, tabular and graphical representations of figures and statistics which can be used to track the current state of the pandemic.

Statitics and Information used within this service have been made publicly available by Public Health England (PHE), The Department for Health and Social Care (DHSC) and The NHS API developer portal.
